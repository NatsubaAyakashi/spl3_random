# Splatoon 3 Random Weapon Bot

## 概要

このBotは、Splatoon 3に登場するブキをランダムに選出するDiscord Botです。
スラッシュコマンド `/random_weapon` を使用して、条件に合ったブキをランダムに一つ選び、そのブキの情報を画像付きで表示します。

## 主な機能

-   **ランダムなブキの選出**: `/random_weapon` コマンドでブキをランダムに選びます。
-   **条件指定**: ブキの種類、サブウェポン、スペシャルウェポンを条件として指定できます。
-   **オートコンプリート**: 条件を指定する際に、入力候補がサジェストされます。
-   **画像表示**: 選ばれたブキの画像がEmbedメッセージに表示されます。

## セットアップ方法

### 1. リポジトリのクローン

```bash
git clone <repository_url>
cd spl3_random
```

### 2. 依存関係のインストール

必要なライブラリをインストールします。

```bash
pip install -r requirements.txt
```

### 3. 環境変数の設定

プロジェクトのルートディレクトリに `.env` ファイルを作成し、お使いのDiscord Botのトークンを記述します。

```
DISCORD_BOT_TOKEN=ここにあなたのBotトークンを記述
```

## 実行方法

以下のコマンドでBotを起動します。

```bash
python main.py
```

## 使い方

### `/random_weapon` コマンド

サーバーのテキストチャンネルで以下のコマンドを入力します。

```
/random_weapon
```

#### オプション

-   `weapon_type`: ブキの種類（シューター、チャージャーなど）を指定します。
-   `sub`: サブウェポン（スプラッシュボムなど）を指定します。
-   `special`: スペシャルウェポン（ウルトラショットなど）を指定します。

**例:**

-   シューターの中からランダムに選出
    ```
    /random_weapon weapon_type:シューター
    ```
-   サブウェポンが「カーリングボム」のブキの中からランダムに選出
    ```
    /random_weapon sub:カーリングボム
    ```
